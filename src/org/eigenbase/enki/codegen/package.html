Contains a code generation infrastructure which is extended by other packages to produce various types of metamodel implementation.

Invoke a concrete {@link org.eigenbase.enki.codegen.Generator} instance via it's main method or, preferably, via the {@link org.eigenbase.enki.ant.MapJavaSubTask} sub-task of the {@link org.eigenbase.enki.ant.EnkiTask Enki ant task}.
