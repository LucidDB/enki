Provides performance-related tests for metamodel implementations. These tests are not run as part of the normal unit test suite.

Note that any test cases placed in this package must be compatible with any implementation that Enki provides for implementing a metamodel. Implementation (e.g. Hibernate or MDR) specific tests should be place elsewhere.
